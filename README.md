## Installing / Getting started

Com o git instalado faça um clone do repositório na máquina local pelo comando:

```shell
git clone https://github.com/FelipeMaxRosa/tinnova-teste-frontend.git
cd tinnova-teste-frontend/
npm install
npm run start
```
